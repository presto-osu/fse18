Description

Note: Please wait couple minutes while the program is installing to wearable via Bluetooth.
IMPORTANT: Please start the app on the smartphone for the first time after installation. It needs to register some services for correct operation.

This is a dialer app (phone calling app aka hand dialer, not voice only) for Android Wear (a wear phone).
Capabilities:
- Contacts list.
- Favorites list.
- Supports multiple phone numbers per contact.
- Call log.
- Keypad.
- Speakerphone on/off (can be auto on when calling from the watch).
- Mic mute/unmute.
- Switching to Bluetooth and back while in call.
- Voice search in the contacts list.
- Dark and light visual themes.
- Customizable startup screen.

Supports round as well as square screens.

If you'll find a bug please let me know.
Feel free for features requests!

P.S. If you are unhappy with your purchase you can cancel it. I will appreciate if you will comment the reason of the cancellation and I will try to implement features which you expected from my app. Thanks!

Also please do not forget to rate this app if you like it!
