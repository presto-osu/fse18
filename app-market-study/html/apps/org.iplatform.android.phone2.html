Description

Realtime Speech Translation. It was showcased on Japanese TV programs.
- Supporting 16 Languages such as English, Japanese, French, German and etc.
- If you call phone to another with this app, you speak your own lang (e.g.Japanese), your voice changing to receiver's own lang (e.g.English).
- New Feature "Bookmark": Registering and Viewing your favorite sentences.
- Fixed Android 4.x Crash problems at opening settings.
- Compatible with Smartphone, Table, Android Wear, Chromecast and SmartEyeglass.
- AI Feature (e.g. "Please change to French" -> Translation will be changed to French.).

Here are 4 modes.

(1)Talk Mode
This mode is intended for translating on the spot.

(2)Phone Mode
This mode is intended for translating over telephone calls.

(3)HandFree Mode
This mode is intended for translating without watching Smartphone Screens.

(4)Camera Mode
If you take pictures, the characters will be automatically translating.

Also it supports Android Wear, Chromecast, SmartEyeglass and etc.

*Notice
If you have installed older version and failed to update app, please try uninstall older app and install new one.
