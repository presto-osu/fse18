Description

Wi-Fi Rabbit is a widget that shows WiFi state by using icon.

It can also show Wi-Fi access point information, toggling Wi-Fi ON/OFF, and switch access point.

Widget style can be selected from "rabbit" or "cat".

It support wearable devices like smartWatch3.

Please check it out "Bluetooth Bear", "Volume Dog", and "3G Monkey".
