Description

Control your foc.us v2 with the official foc.us app.

The foc.us v2 is a transcranial stimulator than can generate electrical waves for tDCS (transcranial direct current stimulation), tACS (transcranial alternating current stimulation), tPCS (transcranial pulsed current stimulation) and tRNS (transcranial random noise stimulation).

This app lets you change the settings for each program.

These are the variables you can change. Please note some settings are mode dependent e.g. no frequency setting for direct current mode.

Mode: tDCS, tACS, tPCS, tRNS
Current: 0-2mA
Time: 1 to 40 mins
Frequncy: 0 to 300Hz
Offset: 0 to 1.5mA
Bipolar: Off, On
Sham: Off, On, Blind
Sham period: 0 to 60 seconds
Voltage: 0 to 60V

You can learn more about foc.us at www.foc.us

*** Please ensure you update your v2 firmware to at least 1.8 via the computer application which can be found at http://www.foc.us/v2/downloads ***

**** If you are running Android 5.0* you will need to upgrade the firmware of your v2 to 1.9. This enables you to turn off bluetooth security as a workaround to a known ble bug in android.
