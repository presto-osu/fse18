Description

This application is an alternative to the official Iris mobile application. This app requires that you have an Iris connected home system & sign up at https://www.irisbylowes.com. This application is in no way affiliated with Lowes or Iris by Lowes and is provided without any warranty. Because of this, it is possible that this application may stop working at anytime there is a change of Iris backend by Lowes. This is one of the risks of using a third party application which provides functionality without an official API by the service provider.

Note: This application works with Iris version 2.0.

Free version lets you view the dashboard and history, control your switches locks, thermostat and irrigation devices. Paid version unlocks all of the other features below:

Devices with temperature, status and more
Presence, including ability to use your mobile device as a keyfob (requires Lowes Premium service)
Get notified when devices disconnect
Get notified when devices have low battery
Get notified when your hub firmware is updated
Get notified when Iris platform is upgraded
Get notified when device signal goes to 0
Custom rules triggered by IFTTT or Tasker (requires Lowes Premium service to run Scenes)
Widgets
Voice Control
Hub information
Rules (requires Lowes Premium service)
Scenes (requires Lowes Premium service)
Android Wear companion app (voice control)

And more features will be added soon.

If you see any issues in the app, please feel free to contact support@80degreeswest.com.
