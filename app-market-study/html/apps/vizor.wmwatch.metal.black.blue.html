Version 0119R premium from 25 May 2015
-------------------------------------------------------------------

Multifunction Android Wear watch face for WatchMaker Premium v.3.4 or later.


-----------------------------------------------------
!!! --- >>> You will need to install the WatchMaker Premium v3.4 or higher in order to use this watch face: http://goo.gl/FMxUfY
-----------------------------------------------------


Features:
- 4 color choices;
- adjusting the brightness of the dimmed mode;
- 2 digital clock mode (12/24);
- weather forecast,
- displaying calendar events,
- list events calendar,
- step counter,
- flashlight mode,
- stopwatch,
- battery level,
- etc


!!! --- >>> Attention! This watchface is protected and is not intended for self-editing. Buying this watchface, you agree to the condition.

Besides, you can always use the free version of this watchface: https://play.google.com/store/apps/details?id=vizor.wmwatch.vmetalblack.blue.v0112r

Or download from the site completely open to editing free version: http://vizorwatch.com


Installation:
---------------------------------------
1. Install on your phone WatchMaker Premium: http://goo.gl/FMxUfY
2. Check it works in conjunction with your watch (set to watch any available dial available in WatchMaker Premium)
3. Install from Google Play this watchface (vMetalBlack blue v.XXX)
4. This watchface will appear on the tab "MY WATCHES" of WatchMaker Premium program
5. Select this watchface
6. Set the watchface on your watch, by clicking on the green button "SET WATCHFACE"
7. Ready!


Enjoy using :)



Update history:
------------------------------------------------------

0112R ---> 0115R 12.05.2015
- significantly enhanced brightness dimmed mode (except the LG G Watch R);
- switch on the screen of the weather forecast is now done by tapping on the icon of the weather (due to the expected changes in future versions WatchMacker 3.7);
- fixed minor bugs.

If you want to increase the brightness in dimmed mode of LG Watch R, the basic program settings WatchMacker specify that you watch Sony Smart Watch 3, for example. If you specify a Moto 360, then you will shift button closure extra screens.

0115R ---> 0116R 14.05.2015
- the flashlight mode is now activated by clicking on a separate icon, but not the logo;
- on the main screen swapped weather data and a digital clock (due to the expected changes in future versions WatchMacker 3.7);
- now switch to the screen of the weather forecast is great as the weather icon and current temperature, and not only the icon of the weather;
- a digital clock on the main screen will now display the time in 12-hour format;
- changed the date format on additional screens (time is still displayed in 24-hour format);
- minor improvements.

0116R ---> 0117R 17.05.2015
- added the ability to choose from four optional colors clearance of the main screen;
- removed the second digital clock from the main screen;
- added setting of time format for the digital clock on the main screen (12 or 24 hours);
- added options for brightness dimmed mode;
- minor improvements.

- indicators "charging", "disconnection" and so on, moved to the center of the screen;
- optimization;
